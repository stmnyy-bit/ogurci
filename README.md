# Практика по SQLite в Python

Проект содержит примеры работы с SQLite в Python:
- подключение к базе данных
- создание таблиц
- вставка, обновление и удаление данных
- выполнение SQL-запросов
- обработка результатов
- транзакции
- prepared requests, views, triggers, indexes
- пример приложения

## Запуск
python main.py
