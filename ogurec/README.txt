Готовая версия приложения для работы напрямую с ogurec.db.

Что важно:
1. ogurec.db уже вложен в архив.
2. Программа автоматически ищет ogurec.db:
   - рядом с program.py
   - внутри папки pyqt_ogurec_app
   - в текущей папке запуска
3. На другом ПК достаточно:
   - распаковать архив
   - установить PyQt5
   - запустить program.py

Запуск:
python program.py
